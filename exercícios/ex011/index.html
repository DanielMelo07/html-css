Diferentes tamanhos

Quando se desenvolve um site, deve-se sempre se preocupar com que dispositivo o usuário vai acessar a página. Desta forma, use a tag "<picture>" para a imagem ser redimensionada de acordo com o tamanho da tela do cliente, por isso, sempre tenha 3 opções de imagem.

Dentro da tag "<picture>", use a tag "<img>" para colocar a imagem, depois use "<source>" e selecione a que tem "media-type", para configurar a forma que ela irá se redimensionar. Nesse sentido, modifique a tag colocando o "media="(min-width: )" para "media="(max-width: 1050px)",assim, se o navegador do cliente tiver menos de 1000 pixels, ele usará a outra imagem, de caráter menor. Porém, vale ressaltar, que são 1000px por que a imagem maior que temos é de 1000, e sempre coloque 50 pixels a mais na tag. Nesse interím, coloque a foto que deve ser substituida no "srcset=" "" e coloque o tipo da imagem no "type="image/". Veja o exemplo.

OBS:

  • Nunca salve o nome das imagens com espaço;
  • Nunca inverta os sources, os maiores são sempre embaixo;
  • Escreva o nome dos documentos corretos para não ter erro;
  • Corte as imagens de acordo com tamanho.

Áudios

Para adicionar áudios ao site, deve-se usar a tag "<audio>" mais o parâmetro "autoplay", para ele poder tocar, e "controls", para aparecer o player no site.
