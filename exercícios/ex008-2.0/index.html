Outros tipos de formatações

Código-Fonte

Para usar a fonte relacionada à códigos, basta usar a tag "<code>".

Ex
Sem a tag:document.getElementByld('teste')
Com a tag:document.getElementByld('teste')

Citações

Para colocar as frases entre aspas, basta usar a tag "<q>".

Ex
Como disse Franz Kafka: há esperança, mas não para nós.

Citações Completas

Para usar uma citação completa, de modo que fique explícito que se trata de uma, basta usar a tag "<blockquote>".

Ex
Albert Camus, em "O mito de Sísifo", diz:

Só existe um problema filosófico realmente sério: é o suicídio. Julgar se a vida vale ou não a pena ser vivida é responder à questão fundamental da filosofia. O resto, se o mundo tem três dimensões, se o espírito tem nove ou doze categorias, aparece em seguida. São jogos. É preciso, antes de tudo, responder. E se é verdade, como pretende Nietzsche, que um filósofo, para ser confiável, deve pregar com o exemplo, percebe-se a importância dessa resposta, já que ela vai preceder o gesto definitivo. Estão aí as evidências que são sensíveis para o coração, mas que é preciso aprofundar para torná-las claras à inteligência.

Abreviações

Quando usar abreviações, use a tag "<abbr>" para mostrar, a medida em que o cursor passa por cima, o nome completo

Ex
Estamos no IBGE.

Texto invertido

Para inverter o texto, basta usar a tag "<bdo>" e selecionar "rtl" para funcionar da direita à esquerda.

Ex
Sem a tag: Eu não dormi direito na semana passada.
Com a tag: Eu não dormi direito na semana passada.
