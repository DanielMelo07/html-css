Links e Hipertextos

Links externos

Para criar um hipertexto, basta usar a tag "<a href="">" e colcocar o link em "href=" """.

Ex: Por aqui, você pode acessar o globo esporte.

Para o link externo abrir em outra aba, basta usar o parâmetro "target="_blank" e rel="external".

Caso o link seja duvidoso, use rel="nofollow"

Ex: Por aqui, você pode acessar o globo esporte.

Links internos

Para criar um hipertexto com um link interno, basta usar a tag "<a href="">" e colcocar o nome do arquivo em "href=" """. Desta forma, não se esqueça de colocar os parâmetros: "target="_blank"" e "rel=" ""

Quando for usar os parâmetros, se estiver na página principal, use o "rel="next"", se estiver na posterior e quiser voltar para a anterior, use "rel="prev"". Assim, o navegador pode entender melhor para onde você deseja ir, por isso, sempre use esse parâmetro.

Ex: Clique aqui, para ir para o meu outro site.

Em outros arquivos

Use as mesmas tags, porém, use ctrl+espaço para selecionar o arquivo do outro site na pasta.

Ex:Clique aqui, para ir para o meu terceiro site.

Caso queira voltar para a página principal, é obrigatorio usar "../" antes de colocar o arquivo desejado, pois signifa que está voltando arquivos para achar o outro.

Ex: "href="../index.html""
