Vídeos

Vídeos em hospedagem própria

Para colocar vídeos, basta usar a tag "<video>". Desat forma, use os parâmetros: "width"" para redimensionar na tela, "controls para ativar os controles e "poster" para colocar uma tumb.

OBS: Está é a forma mais simples

Para fazer da forma mais completa, use a mesma tag, mas com o "source=src" para adicionar o vídeo (Como no caso dos áudios).

OBS: Coloque todos os formatos de vídeo, pois nunca se sabe de que navegador o site será acessado.

OBS: Não coloquei outros formatos, muito menos uma tumb, por ser apenas um teste.

Vídeos sem hospedagem própria

Sabe-se que os videos com hospedagem própria, consomem muito armazenamento de hospedagem do site de forma desnecessária. Nesse sentido, utilize sites de serviços de vídeos para utilizar na página, como o youtube. Para isso, basta selecionar o vídeo, ou postar um próprio, ir em "compartilhar" e utilizar a opção "incorporar", logo após, copiar e colar no site.

Caso queira que o seu vídeo se limite apenas ao seu site, pode-se postar os vídeos no "vimeo", ou até mesmo pegar os vídeos de lá (é o mesmo processo do youtube).
