Listas Ordenadas

Para criar listas ordenadas, ou seja, que a ordem na qual são expostas é de extrema importância, basta usar a tag "<ol>". Nesse sentido, também deve-se usar a tag, dentro da outra tag, "<li>" para colocar os componentes da lista de forma ordenada. Usarei uma lista de ingredientes para bolo como exemplo.

Lista de ingredientes para bolo gelado de coco

  1. 4 ovos
  2. 2 xícaras de chá de farinha de trigo
  3. 1 xícara de chá de açúcar
  4. 1 xícara de chá de leite
  5. 2 colheres de sopa de margarina ou manteiga sem sal
  6. 2 colheres de sopa de coco ralado para a massa
  7. 1 colher de sopa de fermento em pó

Pode-se usar o parâmetro "type", dentro da tag "<ol>", para definir o tipo de lista.

Listas não ordenadas

Para criar listas não ordenadas, ou seja, que a ordem na qual são expostas não tem importância, basta usar a tag "<ul>". Um exemplo seria uma lista de compras.

Lista de Compras

  • Café;
  • Cereais;
  • Achocolatado;
  • Pão;
  • Suco ou chá;
  • Adoçante ou açúcar;
  • Torradas;
  • Geleia.

Junção de listas

Lista de definição

Para criar uma lista de definição, ou seja, uma lista com o componente mais o seu significado, basta usar a tag "<dl>". Nesse sentido, deve-se usar a tag "<dt>" para colocar o termo e a "<dd>" para colocar a descrição.

Palavras e seus significados

Absorto
voltado para os próprios pensamentos; distraído, alheado, abstraído
Inefável
que não se pode nomear ou descrever em razão de sua natureza, força, beleza; indizível, indescritível
Algoz
carrasco, executor da pena de morte ou de outras penas corporais (como tormentos, açoites etc.)
infortúnio
má fortuna; adversidade, desdita, infelicidade
lúgubre
relativo à morte, aos funerais; que evoca a morte; fúnebre, macabro
âmago
a parte mais íntima ou fundamental; essência
Findável
cujo fim é possível ou provável; transitório, finito, efêmero
Infindável
que não pode ter fim ou que parece não ter fim
